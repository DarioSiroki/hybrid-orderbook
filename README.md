# Hybrid orderbook

This project is an implementation of a hybrid orderbook using Python and Spade.

## Running the project

```sh
# Start the agent which emulates external exchange and it's pricing
./okolina.py
# Start the orderbook
./mjenjacnica.py
# Start the market bot
./market_bot.py
# Start as many clients as you wish
./klijent.py <id>
```
